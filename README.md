🌱 IoT Smart Agriculture System 

This project is an IoT-based Smart Agriculture system designed to monitor environmental parameters such as soil moisture, temperature, and humidity in real time

The system aims to help farmers make data-driven decisions by providing continuous monitoring and insights, improving crop productivity and resource efficiency

Features:
- Real-time sensor data monitoring
- Soil moisture, temperature, and humidity tracking
- Automated irrigation control (simulation-based)
- User-friendly dashboard interface
- Data visualization and analysis

Enhancements:
- Improved UI and structure
- Optimized code organization
- Additional monitoring features (to be extended)
